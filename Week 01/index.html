Ever wanted to learn how to code and build websites for yourself or others? This site will introduce you to the basics of how the web works, and how it interprets the code developers use to display intriguing and dynamic websites.

How the Web Works

In its most basic form, when you enter a web address into your browser (e.g. https://www.beginningwebdesign.com), an IP address associated with that website will serve up the "content" of the site back to your browser. At a minimum, this content is in the form of Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS).

What is HTML?

  • HTML stands for Hyper Text Markup Language.
  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page.

HTML consists of a series of elements that encapsulate text data within "tags". HTML elements tell the browser how to display the content on the Web page. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

What is CSS?

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes. CSS saves a lot of work and can control the layout of multiple web pages all at once.

HTML was NEVER intended to contain tags for formatting a web page! HTML was created to describe the content of a web page. To solve this problem, the World Wide Web Consortium (W3C) created CSS.

When writing CSS, developers typically save their style definitions in an external .css file. With an external stylesheet file, you can change the look of an entire website by changing just one file!
